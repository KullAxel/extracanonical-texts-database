How to use this platform

1. Browse & Search

Use the sidebar to filter by category or search across all documents, summaries, and translations.

2. Parallel Versions

Click a document to open the study view. Use the tabs to switch between Literal, Simple, and Original (when available).

3. Study Tools

Metadata shows canonical status across traditions, related texts, sources, and scholarly notes. Cross-references are linked where available.
