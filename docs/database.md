# Extracanonical Jewish and Christian Texts Database, v0.3-local

Living scholarly catalog. Entry-level details are normalized from v0.1 and partly enriched; not a claim of exhaustive coverage.

The project tracks the best scholarly English translation for each text and provides project text slots for future search/AI ingestion. Full-text hosting requires rights clearance or public-domain/open-license confirmation per text.
